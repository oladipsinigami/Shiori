A2A / A2MCP HTTP surface

Native OKX A2A runs through the okx-a2a XMTP daemon (use scripts/start-a2a-daemon.ps1 with the Claude shim so jobs hit this brain). These HTTP endpoints make Shiori callable by agents and tools today.

GET /.well-known/agent.json → agent card
POST /a2a/tasks → { "message": "...", "userId": "..." }
POST /a2a (JSON-RPC) → method "message/send"
GET /a2mcp/tools → tool list
POST /a2mcp/invoke → A2MCP-style tool call
POST /chat → simple chat API
GET /health → liveness
